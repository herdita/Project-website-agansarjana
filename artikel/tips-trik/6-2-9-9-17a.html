Asiknya Bermain Mobile Legend Menggunakan Joystik atau Keyboard

Ok gaes.... :D Kali ini agansarjana akan berbagi, asiknya bermain mobile legend menggunakan joystik atau keyboard

  1. Install emulator nox app player dan game mobile legend terlebih dahulu

    Pelajari sebelumnya :

    Cara Install game mobile legend di pc menggunakan nox app player

  2. Setelah itu jalankan game mobile legendnya (pastikan PC agan terhubung dengan internet yah..)

  3. Lalu kita akan mengaturnya ketika game berjalan, agar lebih mudah penempatan tombolnya, Untuk awalnya kita akan mensetting-nya dengan menggunakan keyboard.

    Agan bisa mengaturnya seperti gambar dibawah ini... ane udah kasih petunjuk dengan angka dan garis.

    Sehingga setelah selesai di configurasi, agan bisa bermain menggunakan keybord.

  4. Bagaimana cara menggunakan joystik ? sebelumnya agan harus mengikuti langkah di atas terlebih dahulu. Setelah itu download aplikasi Joytokey... agan bisa mendownloadnya disini.

    Disana ane sudah settingkan joytokeynya . So, agan bisa menggunakannya atau mengeditnya sesuai tombol yang agan sukai.

    Joytokey itu funsinya, untuk menyesuaikan tombol keyboard dengan tombol joystik agar tombol keybord bisa digunakan di tombol joystik... ngerti gx penjelasannya.

    Ok kalau kurang faham. Maksudnya tombol keyborad bisa di gantikan di tombol joystik, jadi mengetik huruf a di keyborad itu bisa dilakukan di tombol x joystik.

    jadi game apapun, agan bisa menggunakan joystik dengan mengunakan joytokey, tapi gx semuanya game sih, kecuali game yang sudah ada settingan joystik di dalam gamenya.

  5. So setelah di download. Buka aplikasi joytokey nya... buka melalui Run Administrator (klik kanan => run administrator) dan Kita bisa mengaturnya sesuai yang kita inginkan.

    Cara configurasinya cukup mudah.. tinggal menyesuaikan antara tombol keyboard dengan tombol joystik

Sekitan pembahasannya. kalau ada yang perlu ditanyakan silahkan comment di cbox...
